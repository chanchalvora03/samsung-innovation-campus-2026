# Samsung Innovation Campus 2026

This repository contains the work I completed during the Samsung Innovation Campus program. It includes daily notebooks, practice problems, and the two mini projects done towards the end.

The idea was to keep everything in one place as I progressed through the course, instead of treating each part separately.


## What I worked on

The course starts with Python basics and gradually moves into data structures, algorithms, and data analysis.

Some of the main areas covered:

- Python fundamentals and problem solving  
- Stacks, queues, and hashing  
- Searching and sorting algorithms  
- Greedy, recursion, and dynamic programming  
- Pandas for data handling  
- Basic data visualization  


## Structure

The repository follows the same flow as the course:

- Each chapter has its own folder  
- Inside each chapter, notebooks are organized day-wise  

So it’s more of a timeline than a collection of isolated files.


## Mini Projects

Both projects are in `chapter-8-mini-projects`.

### Financial Data Analysis
Worked with different asset classes and explored returns, volatility, and correlations. Also looked at how they behaved during the COVID period.

### Vaccination Data Analysis
Used a global vaccination dataset to study trends and created a simple map visualization using Folium.
